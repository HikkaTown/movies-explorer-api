# movies-explorer-api

Перед запуском проекта рекомендуется установить зависимости с помощью команды:
```sh
npm install
```


## Управление пректом:

Запуск сервера: 
```sh
npm run start
```

Запуск сервера с nodemon:
```sh
npm run dev
```

Запуск eslint: 
```sh
npx eslint .
```
